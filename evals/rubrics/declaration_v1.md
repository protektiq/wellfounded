# Declaration quality rubric v1

Score each criterion on an integer scale from **1** (unacceptable) to **5** (exemplar).
Use **0** only when the criterion cannot be assessed from the materials supplied (explain in reasoning).

Return scores using these exact criterion keys in the `criteria` map:

- `faithfulness_to_source`
- `structural_completeness`
- `voice_authenticity`
- `flag_accuracy`
- `legal_element_coverage`

The overall `score` should reflect the weakest dimension that materially affects filing readiness,
not a simple average. Be exacting and conservative.

## faithfulness_to_source

Does the draft assert only facts supported by the interview transcript and prior statements?

| Score | Anchor |
|-------|--------|
| 1 | Material facts invented, contradicted, or unsupported by source |
| 2 | Several unsupported assertions or silent distortion of key events |
| 3 | Mostly faithful with minor unsupported details |
| 4 | Faithful; only trivial embellishment |
| 5 | Every material assertion traceable to source segments or priors |

## structural_completeness

Are all required declaration sections present and substantive (not placeholder boilerplate)?

Required sections: identity_background, past_persecution, perpetrator_motivation,
well_founded_fear_future, internal_relocation, filing_bar_facts.

| Score | Anchor |
|-------|--------|
| 1 | Multiple required sections missing or empty |
| 2 | Sections present but largely skeletal or generic |
| 3 | All sections present; one or two thin |
| 4 | All sections substantive with minor gaps |
| 5 | All sections complete and case-specific |

## voice_authenticity

Does the declaration read in the client's first-person voice rather than stilted legalese?

| Score | Anchor |
|-------|--------|
| 1 | Reads as template or attorney voice throughout |
| 2 | Frequent formal boilerplate; little client voice |
| 3 | Mixed voice; acceptable but uneven |
| 4 | Predominantly natural client voice |
| 5 | Consistent, credible first-person narrative |

## flag_accuracy

Are GAP, INFERENCE, and INCONSISTENCY flags placed correctly and substantively?

| Score | Anchor |
|-------|--------|
| 1 | Major gaps or inconsistencies missed; false flags dominate |
| 2 | Several misplaced or missing flags |
| 3 | Adequate flag coverage with some errors |
| 4 | Flags mostly correct and actionable |
| 5 | Flags precise, well-placed, and substantively described |

## legal_element_coverage

Are protected ground, nexus, and well-founded fear explicitly addressed for this claim?

| Score | Anchor |
|-------|--------|
| 1 | Core legal elements absent or conflated |
| 2 | Elements mentioned but not developed |
| 3 | All elements present; weak nexus or fear analysis |
| 4 | Clear coverage of ground, nexus, and fear |
| 5 | Exemplar articulation tied to client-specific facts |
